# Pizza_Sales_Dashboard
This project showcases an interactive Pizza Sales Dashboard developed using Microsoft Excel, with all KPIs and insights validated through SQL queries in SQL Server Management Studio (SSMS). The analysis is based on a real-world pizza sales CSV dataset and focuses on extracting meaningful business insights from raw transactional data.

**KPI Analysis:-** 

1.Total Revenue
    
2.Total Orders
    
3.Average Order Value
    
4.Total Pizzas Sold
    
5.Average Pizzas per Order

**Trend Analysis:-**

1.Daily and hourly order trends
    
2.Busiest days and peak order times

**Product Insights:-**

1.Sales distribution by pizza category and size
    
2.Best and worst-selling pizzas
    
3.Category & size-wise maximum sales analysis

**Interactive Dashboard (Excel):-**

1.Dynamic charts and slicers
    
2.Clean visual storytelling for decision-making

**SQL Validation:-**

1.All KPIs and metrics are cross-verified using SQL queries in SSMS to ensure accuracy and data integrity
SQL Query Document:- [SQL Queries](./pdf/SQL_Queries_in_SSMS.pdf).
